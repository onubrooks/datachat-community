# DataChat Community (SQL-First)

DataChat Community is the open source, SQL-first edition of DataChat focused on natural-language-to-SQL and direct SQL workflows.

## Status

Bootstrap repository created. Community code and docs are being prepared.

## Planned Scope

- SQL editor (read-only execution)
- Natural-language query flow for database analytics
- Result rendering (table + chart + evidence)
- Basic database setup and schema exploration

## Not Included

- Advanced orchestration/action-loop runtime
- Proprietary workflow packs and enterprise automation
- Internal roadmap/specs for dynamic harness features
