# DataChat Community

DataChat Community is the open source edition of DataChat focused on reliable data querying, onboarding, and managed DataPoint workflows.

## Status

Bootstrap repository created. Community code and docs are being prepared.

## Community Scope

- SQL editor mode (read-only execution)
- Natural-language query flow for database analytics
- Current reliable harness/runtime path (as shipped)
- Result rendering (answer/sql/table/visualization/evidence/timing)
- Onboarding wizard (UI + CLI)
- Managed DataPoint generation/editor/review workflows
- CLI equivalents for setup/query/onboarding/datapoint sync

## Not Included

- Proprietary workflow packs and enterprise-only automation layers
- Internal strategy docs, private roadmap/specs, and internal eval artifacts
